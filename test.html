Window Inner, Outer Height:
The Window.outerHeight read-only property returns the height in pixels of the whole browser window, including any sidebar, window chrome, and window-resizing borders/handles.

Document Client Height:
The inner height of an element in pixels, including padding but not the horizontal scrollbar height, border, or margin

Document Body Offset Height:
A measurement which includes the element borders, the element vertical padding, the element horizontal scrollbar (if present, if rendered) and the element CSS height.
